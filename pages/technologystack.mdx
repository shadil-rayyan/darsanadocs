# Technology Stack

Our project is built using a modern and robust technology stack designed for efficiency, scalability, and maintainability. Below is an overview of the key technologies and tools we are leveraging:

## Frontend

- **Next.js**: A React framework that enables server-side rendering and static site generation, providing a high-performance and SEO-friendly experience.
- **Tailwind CSS**: A utility-first CSS framework that allows for rapid and flexible styling directly within your markup.
- **TypeScript**: A strongly typed programming language that builds on JavaScript, enhancing code quality and developer productivity.

## Backend

- **Vercel Postgres**: A fully managed SQL database service that offers scalable and reliable data storage.
- **Drizzle ORM**: An Object-Relational Mapping (ORM) tool for Vercel Postgres, facilitating smooth interactions between our application and the database.

## Authentication and Media

- **Firebase**: Provides authentication services and image hosting, simplifying user management and media handling.

## Development Tools

- **GitHub**: A platform for version control and collaboration, enabling efficient code management and team collaboration.
- **Git**: A distributed version control system for tracking changes in source code during development.
- **Docker**: A containerization platform that ensures consistent development environments and smooth deployment.
- **Husky**: A tool for managing Git hooks, helping enforce code quality and consistent commit messages.
- **Lint-staged**: Runs linters on pre-committed files, ensuring code quality before changes are committed.

## Code Quality and Formatting

- **ESLint**: A linting tool for identifying and fixing problems in JavaScript and TypeScript code.
- **Prettier**: An opinionated code formatter that ensures consistent code style across the project.
- **Commitlint**: A tool for enforcing commit message conventions, maintaining consistent and meaningful commit history.

## Testing

- **Jest**: A testing framework for running unit and integration tests, ensuring code reliability and functionality.
- **Playwright**: A testing library for end-to-end testing of web applications, ensuring they work as expected in different browsers.
